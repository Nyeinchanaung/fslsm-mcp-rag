A post-hoc pairwise evaluation was conducted over all 5,760 session pairs
from Experiment 2 using GPT-4o as LLM-as-a-Judge. For each pair, the judge
was presented with the same student question, the agent's FSLSM profile
description, and both R0 and R1 responses (fixed order, A=R0/B=R1), and
asked to select the more pedagogically appropriate response.

R1 (personalized) responses were preferred in 94.1% of decisive comparisons
(n=5,760, ties excluded), compared to 5.9% for R0. A one-tailed binomial
test confirmed this preference was statistically significant (p < 2.2e-16,
95% CI 0.934–0.947). The tie rate was 0.0%. Per-profile analysis showed R1
winning in 16 of 16 FSLSM profiles at Bonferroni-corrected significance
(α=0.003), with per-profile win rates ranging from 79.4% (Reflective-
Sensing-Visual-Sequential) to 100.0% (Active-Sensing-Verbal-Global).

Convergent validity (Spearman correlation between per-question pairwise win
rate and per-question Likert Engagement/SCS delta) could not be computed
because the pairwise win rate was near-constant across questions (median
~94%), yielding no discriminating variance. This uniformity is itself a
substantive finding: R1's pedagogical advantage was consistent across
question content, not confined to a subset of questions.
