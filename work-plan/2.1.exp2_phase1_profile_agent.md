## Goal

Translate a binary FSLSM vector `[-1|+1, -1|+1, -1|+1, -1|+1]` into a natural-language **reasoning plan** that downstream agents (Retrieval Agent, Tutor Agent) use to shape retrieval and generation. This is the first component of the PersonaRAG pipeline.

## Input / Output Contract

```python
# Input
fslsm_vector: dict = {
    "act_ref": -1,   # -1=Active, +1=Reflective
    "sen_int": -1,   # -1=Sensing, +1=Intuitive
    "vis_ver": -1,   # -1=Visual, +1=Verbal
    "seq_glo": +1    # -1=Sequential, +1=Global
}

# Output
reasoning_plan: dict = {
    "profile_code": "P02_ActSenVisGlo",
    "style_label": "Active-Sensing-Visual-Global",
    "retrieval_directive": "Retrieve concrete worked examples with diagrams; prioritize conceptual overviews before step-by-step details.",
    "generation_directive": "Begin with the big picture. Use visual scaffolding (diagrams, charts). Ground in concrete facts. Encourage hands-on thinking.",
    "reranking_bias": ["visual_content", "conceptual_overview", "concrete_examples"],
    "deprioritize": ["abstract_theory", "step_by_step_only"],
    "style_descriptor_graf": "Prefers hands-on, concrete, visual instruction with an overview-first structure..."
}
```

## Tasks

### Task 1.1 — Define `FSLSM_STYLE_MAP`

A lookup table mapping each of the 16 binary FSLSM combinations to retrieval/generation directives.

```
[ ] 1.1.1  Create FSLSM_STYLE_MAP as a dict keyed by (act_ref, sen_int, vis_ver, seq_glo) tuples
[ ] 1.1.2  For each of the 16 profiles, define:
           - retrieval_directive: what kind of chunks to seek from FAISS
           - generation_directive: how the tutor LLM should structure its response
           - reranking_bias: list of content tags to boost during FSLSM-aware reranking
           - deprioritize: list of content tags to demote
[ ] 1.1.3  Source directives from Graf et al. (2007) dimension descriptors
           and existing style_descriptor_graf in data/fslsm/profiles.json
```

**Reference: Dimension-to-Directive Mapping (Graf et al., 2007)**

| Dimension | Pole (-1) | Retrieval Bias | Generation Style |
|---|---|---|---|
| act_ref | Active | interactive examples, exercises, hands-on | Encourage experimentation, group activities |
| act_ref | Reflective (+1) | theoretical summaries, case studies | Provide reflection time, analytical structure |
| sen_int | Sensing | concrete facts, worked examples, procedures | Ground in specifics, practical applications |
| sen_int | Intuitive (+1) | abstract concepts, theoretical frameworks | Focus on principles, generalizations |
| vis_ver | Visual | diagrams, charts, spatial representations | Use visual scaffolding (LaTeX, ASCII art) |
| vis_ver | Verbal (+1) | text explanations, narratives, definitions | Use prose, analogies, verbal descriptions |
| seq_glo | Sequential | step-by-step walkthroughs, linear structure | Build concepts one at a time, numbered steps |
| seq_glo | Global (+1) | conceptual overviews, big-picture summaries | Overview first, then zoom into details |

**Example entry:**
```python
FSLSM_STYLE_MAP = {
    (-1, -1, -1, -1): {  # P01: Active-Sensing-Visual-Sequential
        "profile_code": "P01_ActSenVisSeq",
        "style_label": "Active-Sensing-Visual-Sequential",
        "retrieval_directive": (
            "Retrieve step-by-step visual walkthroughs with concrete worked examples. "
            "Prioritize chunks containing diagrams, numbered procedures, and practical exercises."
        ),
        "generation_directive": (
            "Use numbered steps with diagrams and concrete examples. "
            "Encourage hands-on practice. Ground every concept in specific, observable facts. "
            "Build linearly — do not skip steps or jump to the big picture."
        ),
        "reranking_bias": ["visual_content", "step_by_step", "concrete_examples", "exercises"],
        "deprioritize": ["abstract_theory", "conceptual_overview"]
    },
    (-1, -1, -1, +1): {  # P02: Active-Sensing-Visual-Global
        "profile_code": "P02_ActSenVisGlo",
        "style_label": "Active-Sensing-Visual-Global",
        "retrieval_directive": (
            "Retrieve concrete worked examples with diagrams. "
            "Prioritize conceptual overviews and big-picture summaries before details."
        ),
        "generation_directive": (
            "Begin with the big picture — show how this topic connects to the broader framework. "
            "Use visual scaffolding (diagrams, charts). Ground in concrete facts. "
            "Encourage hands-on exploration after establishing context."
        ),
        "reranking_bias": ["visual_content", "conceptual_overview", "concrete_examples"],
        "deprioritize": ["abstract_theory", "step_by_step_only"]
    },
    # ... all 16 combinations
}
```
