## Metric Definitions & Formulas

### 1. Style Conformance Score (SCS)

Measures how well the tutor response aligns with FSLSM pedagogical guidelines.

```
SCS(r, s) = (Vr . Vs) / (||Vr|| x ||Vs||)

Where:
  Vr = embedding of the tutor's text response
  Vs = embedding of the FSLSM style description anchor (Graf et al., 2007)
```

**Expected:** R0 mean ~ 0.45, R1 mean ~ 0.70 (primary hypothesis target)

### 2. Response Relevance (RR)

LLM-as-a-Judge scoring against the D2L ground truth answer.

```
RR = (1/N) x SUM Judge(ri, gi)

Where:
  Judge(*) -> score in {1, 2, 3, 4, 5} (1=Irrelevant, 5=Highly Relevant)
  ri = system response for session i
  gi = gold answer for question i
```

**Expected:** R0 ~ 4.1, R1 ~ 4.2 (should remain stable — personalization must not degrade accuracy)

### 3. Chunk Recall@5 (CR@5)

```
CR@5 = |R intersect G| / |G|

Where:
  R = set of top-5 retrieved chunk IDs
  G = set of gold (ground-truth) chunk IDs
```

**Expected:** R0 ~ 0.42, R1 ~ 0.50

### 4. Essential Recall (ER)

```
ER = |R intersect E| / |E|

Where:
  E = set of essential chunks (annotated as must-have for correct answer)
```

**Expected:** R0 ~ 0.48, R1 ~ 0.55

### 5. Engagement Score (Eng)

Already collected during Phase 4 — self-reported by virtual student agent (1-5).

**Expected:** R0 ~ 3.2, R1 ~ 4.5

**Cohen's d formula:**
```python
def cohens_d(group1, group2):
    n1, n2 = len(group1), len(group2)
    var1, var2 = np.var(group1, ddof=1), np.var(group2, ddof=1)
    pooled_std = np.sqrt(((n1-1)*var1 + (n2-1)*var2) / (n1+n2-2))
    return (np.mean(group1) - np.mean(group2)) / pooled_std
```

## Acceptance Criteria (H1 Confirmation)

The experiment confirms Hypothesis H1 when ALL of these are met:

```
 4,000 R0 sessions logged with no >5% missing metric values
 4,000 R1 sessions logged with no >5% missing metric values
 SCS(R1) > SCS(R0) with p < 0.05 (H1 confirmed — primary result)
 SCS uplift >= 0.20 absolute (R1 vs R0)
 RR(R1) ~ RR(R0) +/- 0.3 (personalization does not degrade factual accuracy)
 CR@5(R1) >= CR@5(R0) (FSLSM-aware retrieval improves chunk coverage)
 Engagement(R1) > Engagement(R0) with p < 0.05
 exp2_results_summary.json is valid JSON with all 5 metrics
```
